# FluentOS 设计规格

> **日期:** 2026-06-22
> **版本:** 1.0.0
> **状态:** 已批准

## 1. 项目概述

**项目名称:** FluentOS

**项目类型:** 基于 Arch Linux 的轻量级定制桌面操作系统

**核心定位:** 高度Windows操作逻辑兼容的Linux发行版，融合Fluent+液态玻璃设计语言

**目标用户:**
- 技术爱好者/开发者
- Windows普通用户（需无缝过渡）
- 企业/教育环境

---

## 2. 技术架构

### 2.1 基础系统

| 组件 | 技术选型 | 说明 |
|------|----------|------|
| 基础发行版 | Arch Linux | 滚动更新、轻量、可定制 |
| 内核 | Linux Kernel (最新稳定版) | 默认6.x |
| init系统 | systemd | Arch默认 |
| 包管理器 | pacman + AUR (yay) | 轻量高效 |

### 2.2 桌面环境

| 组件 | 技术选型 | 说明 |
|------|----------|------|
| 桌面环境 | KDE Plasma 5 (高度定制) | 可在高性能模式下切换到 XFCE |
| 窗口管理器 | KWin (默认) / Openbox (轻量模式) | 可切换 |
| 文件管理器 | Dolphim (Fork自Dolphin) | 仿Windows资源管理器 |
| 任务栏/开始菜单 | 自定义 widget + Latte Dock | 仿Windows体验 |
| 主题引擎 | Qt5/Qt6 | Fluent Design 实现 |

### 2.3 特效系统

| 组件 | 技术选型 | 说明 |
|------|----------|------|
| 窗口特效 | KWin + 自定义 GLSL shader | 液态玻璃、模糊、阴影 |
| 动画引擎 | Qt Quick/QML | 60fps 流畅动画 |
| 默认状态 | **性能模式（特效禁用）** | 降低CPU/内存占用 |
| 用户可选 | 高性能模式（特效启用） | 需用户确认警告 |

### 2.4 Windows 兼容性

| 组件 | 技术选型 | 说明 |
|------|----------|------|
| 文件系统共享 | ntfs-3g | 读写Windows NTFS分区 |
| 程序调用 | 符号链接 + 启动器 | 直接调用Windows程序 |
| 双系统引导 | GRUB 定制 + rEFInd | 美化引导界面 |

### 2.5 安装程序

| 组件 | 技术选型 | 说明 |
|------|----------|------|
| 安装框架 | Calamares | 模块化、可定制 |
| Windows内安装 | 自定义PE环境 | 无需重启即可安装 |
| Live环境 | Arch ISO 定制 | 试用和安装 |

## 3. 系统要求

### 3.1 最低配置 (性能模式)

| 硬件 | 最低要求 |
|------|----------|
| CPU | Intel Core i3 4代 或 AMD 同等处理器 |
| 内存 | 2GB RAM |
| 存储 | 20GB 可用空间 |
| 显卡 | Intel HD Graphics (集成) |
| 显示 | 1024x768 |

### 3.2 推荐配置 (高性能模式)

| 硬件 | 推荐要求 |
|------|----------|
| CPU | Intel Core i5 4代 或 AMD 同等处理器 |
| 内存 | 4GB+ RAM |
| 存储 | 40GB+ 可用空间 |
| 显卡 | Intel HD Graphics 4000+ 或 独立显卡 |
| 显示 | 1920x1080 |

## 4. 功能优先级

| 优先级 | 功能 | 说明 |
|--------|------|------|
| P0 | Windows操作逻辑兼容 | 文件资源管理器、任务栏、开始菜单、快捷键 |
| P0 | Windows应用生态兼容 | 双系统文件共享，直接调用Windows程序 |
| P1 | 多系统引导支持 | GRUB定制，支持Windows/Linux双启动 |
| P1 | 安装程序 | Windows内安装 + Live USB安装 |
| P2 | Fluent+液态玻璃动画 | 可选的视觉效果 |
| P2 | 低配置优化 | 4代I3可流畅运行 |

## 5. 用户体验设计

### 5.1 特效模式

- **默认：性能模式**
  - 特效禁用
  - 内存占用 < 400MB (空闲)
  - 适合老旧硬件

- **可选：高性能模式**
  - 启用Fluent+液态玻璃动画
  - 弹出警告确认
  - 适合现代硬件

### 5.2 双系统集成

```
FluentOS 文件管理器
├── 主目录 (/home)
├── Windows分区 (/mnt/windows)
│   ├── C:\Program Files → 启动器
│   └── C:\Program Files (x86) → 启动器
└── 网络邻居
```

- Windows程序以启动器形式出现在应用菜单
- 点击自动挂载对应Windows分区并运行程序

## 8. 设计决策记录

| 日期 | 决策 | 理由 |
|------|------|------|
| 2026-06-22 | 基于Arch Linux而非从零构建 | 平衡可控性和工程量 |
| 2026-06-22 | 默认禁用特效 | 确保4代I3可流畅运行 |
| 2026-06-22 | 双系统文件共享而非虚拟化 | 无需Windows授权，保留原生性能 |
| 2026-22 | 支持Windows内安装 | 降低Windows用户迁移门槛 |
